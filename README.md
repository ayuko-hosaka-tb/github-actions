# サンプルアプリケーション

性的なフロントエンドと Web API から成る、足し算アプリケーションです。

## 開発モードでの起動手順

サンプルアプリケーションは、以下の手順で開発モードで起動します。

```
cd backend/
npm run dev
```

起動したら、ブラウザで http://localhost:8080 にアクセスしてください。

## バックエンドをビルドして実行する手順

バックエンドをビルドして実行する場合は、以下のコマンドを実行してください。

```
cd backend/
npm run build
node dist/index.js
```
